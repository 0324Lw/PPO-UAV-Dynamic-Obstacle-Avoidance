# 🚀 PPO-UAV-Dynamic-Obstacle-Avoidance

> 基于 Proximal Policy Optimization (PPO) 深度强化学习的无人机室内微观场景动态避障与路径规划算法。
> 本项目构建了一个高度自定义的 `gymnasium` 强化学习环境，支持混合动静态障碍物避障，并内置了完整的 PPO 训练管线与全自动的数据记录和 SCI 级别绘图系统。

## ✨ 核心特性 (Features)

* 🌍 **定制化微观环境**：构建了 20m x 20m 的室内地图，支持连续动作空间（线速度与角速度控制）。
* 🛸 **高复杂度混合避障**：环境内置 5 个静态障碍物和 3 个动态障碍物，障碍物形状随机生成（圆形或方形）。环境生成逻辑严格保证了起终点距离限制（20m~25m）以及障碍物之间的最小安全间距。
* 🧠 **PPO 强化学习底座**：采用 MLP 结构的 Actor-Critic 网络。集成了广义优势估计 (GAE)、梯度截断 (Clip)、熵正则化 (Entropy Coef) 以及动态学习率衰减等 PPO 核心机制。
* 📊 **运行状态归一化**：内置 `RunningMeanStd` 模块，对无人机获取的 32 维环境观测状态（包含目标距离、相对角度、各障碍物距离与速度等）进行实时在线均值方差归一化处理，极大提升训练稳定性。
* 📈 **全自动数据分析与可视化**：提供完备的 `DataLogger` 接口。训练过程中记录 11 项核心数据，结束后自动保存为 CSV 文件，并利用零相位双重指数平滑算法生成包含成功率、奖励值、损失函数等指标的高清 SCI 级别趋势图表。

## 🧠 强化学习环境设计 (Environment Design)

本项目针对高动态、高密度的微观室内避障场景进行建模。为了让无人机在复杂的动态交互中学习到丝滑的避障策略，核心的状态空间、动作空间与奖励函数设计如下：

### 1. 状态空间 (Observation Space)

智能体的观测状态被精心设计为 **32 维连续特征向量 (Continuous 32D)**，由无人机的核心运动状态与周围环境的感知信息拼接而成。所有输入均经过归一化处理，以便于神经网络高效提取特征：

| 维度索引 | 物理含义 (Description) | 数据范围 / 说明 |
| :---: | :--- | :--- |
| `0:7` | **自身核心状态**：包含归一化的目标距离、相对目标偏航角、线速度 $v$、角速度 $\omega$，以及速度向量与目标方向的夹角余弦值 $\cos(\theta)$ | 包含补齐占位符以对齐维度 |
| `7:32` | **局部动态感知状态**：动态扫描并提取距离无人机**最近的 5 个障碍物**的特征。每个障碍物独占 5 维：<br>1. 边界相对距离<br>2. 相对偏航角<br>3. 相对速度分量<br>4. 障碍物类型标签 (Type)<br>5. 预留维度 | 若视野内障碍物不足 5 个，则自动使用 `[1.0, 0.0, 0.0, 0.0, -1]` 构成的安全特征进行 Padding 补齐 |

### 2. 动作空间 (Action Space)

项目抛弃了非连续的离散动作，采用 **2 维连续动作空间 (Continuous 2D)**，直接控制无人机的底层运动学解算：

* **$a_0$ (线速度控制)**：归一化输出 $[-1.0, 1.0]$，映射至无人机的实际线速度 $v$。
* **$a_1$ (角速度控制)**：归一化输出 $[-1.0, 1.0]$，映射至无人机的实际角速度 $\omega$。

### 3. 奖励函数设计 (Reward Function)

为了在极度拥挤的动态环境中兼顾“敏捷避障”与“平滑控制”，环境设计了极其精细的稠密奖励与动作惩罚机制：

$$R_{step} = r_{step} + r_{fwd} + r_{back} + r_{dir} + r_{smooth} + r_{danger}$$

* 🔴 **时间惩罚 ($r_{step}$)**：每步固定的微小惩罚，促使模型寻找时间最优解。
* 🟢 **趋近/后退奖励 ($r_{fwd}, r_{back}$)**：基于单步距离目标的差值。有效接近目标给予线性正向奖励；若产生远离目标的倒退行为，则给予更严厉的**二次方惩罚**。
* 🟢 **航向引导 ($r_{dir}$)**：基于机头朝向与目标方向的偏差角，偏差越小奖励越高，引导无人机始终“盯”着终点。
* 🔴 **平滑度惩罚 ($r_{smooth}$)**：对相邻两步之间的动作变化率 ($\Delta a$) 施加二次方惩罚，极大地抑制了 PPO 算法早期极易产生的“高频抖动”现象，保护真实电机的寿命。
* 🔴 **危险区斥力 ($r_{danger}$)**：当距离障碍物小于 1.5m 时触发。距离越近，产生的二次方排斥惩罚越大（单步惩罚存在硬性下限保护，防止梯度爆炸）。

**🏆 终局事件与抗梯度爆炸设计 (Reward Clipping)：**
为了保证 PPO 算法的绝对稳定：
1. **常规截断**：所有的连续步奖励 $R_{step}$ 都会被严格限幅 (Clip) 在 **$[-2.0, 2.0]$** 之间。
2. **终局直通**：越界/碰撞极刑 ($r_{collision}$) 与到达终点大奖 ($r_{goal}$) 将**绕过截断机制**直接叠加。这确保了在避免日常探索产生异常梯度的同时，保留了最核心的胜负信号。

## 🛠️ 环境依赖 (Requirements)

本项目基于 Python 开发，核心依赖库如下：

* `torch` (PyTorch)
* `gymnasium`
* `numpy`
* `pandas`
* `matplotlib`
* `scipy`

**快速安装指南 (Conda)：**
```bash
conda create -n uav_ppo python=3.10 -y
conda activate uav_ppo
conda install pytorch torchvision torchaudio cpuonly -c pytorch -y
conda install numpy pandas matplotlib scipy gymnasium -c conda-forge -y
```
